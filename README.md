# CamPhotoService

Простые скрипты для стриминга фото/видео в форматах JPEG/MJPEG через сервер-посредник.

## app_jpegs.py

Сервис для просмотра фотографий, загруженных с удаленной веб-камеры.

Изображения снимаются с камеры каждую секунду **ffmpeg** и загружаются с помощью **curl**.
На странице просмотра можно выбрать режим отображения текущего кадра - "видео",
либо просматривать загруженные кадры по номеру

## app_mjpeg.py

Сервис для приема непрерывного потока кадров JPEG с помощью TCP сервера
и передачи клиентам через HTTP для отображения в формате MJPEG.

Передача потока JPEG с камеры может осуществляться с помощью **ffmpeg**
